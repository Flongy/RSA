# Шифрование RSA
Данная библиотека предоставляет возможность шифрования данных по алгоритму [**RSA**](https://en.wikipedia.org/wiki/RSA_(cryptosystem))  (шифрования с открытым ключем) на языке программирования [**D**](https://dlang.org/), для которого разработана собственная реализация больших чисел.  
### Инструкция по использованию из консоли:  
- `rsa generate num1 num2` - создать новые ключи для шифрования, где **num1** и **num2** - простые числа, на основе которых высчитываются ключи;  
- `rsa encrypt pk text` - зашифровать текст **text** с помощью открытого ключа **pk**;  
- `rsa decrypt pk sk ciphertext` - расшифровать шифротекст **ciphertext** с помощью открытого ключа **pk** и закрытого ключа **sk**.  

`rsa` - имя исполняемого файла (нужно сначала скомпилировать: `dub build`).  

Можно также выполнять шифрование с данными, располагающихся в файлах, для этого нужно добавить в конце `-f`. Например: `rsa encrypt pk.key text.txt -f` - будет зашифрован текст из файла **text.txt**, ключ будет считан из **pk.key**  
### Инструкция по использованию в коде:  
В данной библиотеке есть два модуля: **BigInt** и **RSA**  
#### Модуль BigInt  
Является собственным вариантом представления больших чисел, отличный от [GMP](https://gmplib.org/), и реализованный на чистом [D](https://dlang.org/).  
Для определения BigInt в конструкторе можно использовать целое число (`int` или `long`) или строку, представляющую число в десятичном, шестнадцатеричном (с префиксом `0x`) или двоичном (с префиксом `0b`) виде:  
```D
BigInt var1 = new BigInt(245);    // Конструктор с целым числом
BigInt var2 = new BigInt("245");  // Конструктор из строки с десятичным числом
BigInt var3 = new BigInt("0xF5"); // Конструктор из строки с шестнадцатиричным числом
BigInt var4 = new BigInt("0b1111 0101"); // Конструктор из строки с двоичным числом
```  
Большие числа поддерживают операции: `+`, `-`, `*`, `/`, `%`, `<<`, `>>`, а также возведение в степень и возведение в степень по модулю:
```D
BigInt binum1 = new BigInt(150);
BigInt binum2 = new BigInt(12);

auto binum3 = binum1 ** binum2;                       // Возведение в степень
auto binum4 = BigInt.powMod(binum3, binum2, binum1);  // Возведение в степень по модулю
```
